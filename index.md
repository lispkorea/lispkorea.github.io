# Lisp을 좋아하는 사람들의 그룹(한국 리스퍼)
 `Lisp`에 관련된 체계적인 정보를 모으고, 나누고자 사이트를 개설하였습니다. `Lisp` 관련된 자료를 함께 체계적으로 정리해, 공유해봅시다.

(let [[lisper non-lisper]
      (split-with know-lisp? people)]
